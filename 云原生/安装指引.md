# 安装
## 安装前环境准备
>1. 准备服务器，内网互通
>2. 安装容器化环境(docker)
>3. 安装kubernetes
>> 服务器安装核心组件(kebuadm创建集群引导工具, kubelet, kubectl命令行工具)
>> kubelet可以直接通过容器化方式创建出其他核心组件
>> 由kubeadm引导创建集群

## 服务器环境准备
>1. 设置防火墙打开所需要的端口
>2. 设置hostname
>> `hostnamectl set-hostname k8s-0x`
>3. 设置host映射
>> `echo "127.0.0.1 $(hostname)" >> /etc/hosts`
